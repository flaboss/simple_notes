# Kivy Notes App

Um gestor de notas moderno e minimalista construído com Python e Kivy.

## Tecnologias
- **Framework:** Kivy 2.3.0
- **Linter/Formatter:** Ruff (rápido e moderno)
- **Testes:** Pytest
- **Automação:** Makefile
- **Base de Dados:** JSON Store (Persistência local)

## Como Configurar

### Pré-requisitos (macOS)
```bash
brew install sdl2 sdl2_image sdl2_ttf sdl2_mixer gstreamer
````

### Instalação
Utilize o Makefile para configurar o ambiente virtual e as dependências:
```bash
make install
````

### Execuçãp
```bash
make run
````
